## 개요
Java 언어 및 디자인 패턴을 사용하여 영화관 시스템 제작

## 목표
- 사용 용도에 맞는 다양한 디자인 패턴 사용
- 디자인 패턴의 원리 이해
- 구현 중 문제점 해결
- 구현 설계 변경에 대한 유연한 대처
